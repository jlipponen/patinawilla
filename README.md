# PatinaWilla website

A modern React/TypeScript marketing website for PatinaWilla Oy, an upholstery and restoration company based in Ulvila, Finland.

The site showcases services, portfolio, and company information with a clean and professional design.

Available in [https://patinawilla.fi/](https://patinawilla.fi/)

# Development
## Scripts
- `npm run dev` start dev server
- `npm run build` production build
- `npm run preview` preview production build

## Getting started
```bash
npm install --ignore-scripts
npm run dev
```

Open http://localhost:5173
